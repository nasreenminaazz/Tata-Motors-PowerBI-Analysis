# Financial Performance and Cash Flow Analysis of Tata Motors Using Microsoft Power BI

## Project Overview
This project analyzes Tata Motors' financial performance using Microsoft Power BI.

## Objectives
- Revenue Analysis
- Profit Analysis
- Cash Flow Analysis
- Asset Growth Analysis
- Employee Growth Analysis

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX

## Key Findings
- Revenue increased significantly from FY21 to FY25.
- Profitability improved after FY22.
- Strong operating cash flow was observed.
- Asset growth reflected business expansion.
